# 기획서 초안으로 에디터 수정중

> AI가 프론트엔드 코드 + Figma를 교차 분석하여 자동 생성한 초안입니다.

---

## 서비스 개요

**Ailex** — 법률 AI 플랫폼. 사건 관리, 증거 열람, AI 서면 작성 기능을 제공합니다.

---

## 화면 목록

| 화면명 | 경로 | 주요 기능 |
|--------|------|-----------|
| 로그인 | / | 이메일/비밀번호 로그인 |
| 홈 | /home | 대시보드, 최근 사건 목록 |
| 사건 목록 | /case-list | 사건 목록 조회, 탭 전환 |
| 사건 에디터 | /case-list/editor | 서면 작성 |
| 증거 목록 | /evidence/list | 증거 파일 목록 |
| 증거 PDF | /evidence/pdf/:id | PDF 뷰어 |
| 증거 뷰어 | /evidence/case-viewer/:id | 사건 연결 증거 뷰어 |
| AI | /ai | AI 서면 생성 |
| 설정 | /settings | 계정 설정 |
| 알림 | /notifications | 알림 목록 |
| 메모 | /memo | 메모 작성/조회 |
| 결제 | /payment | 결제 수단 관리 |
| 구독 | /subscription | 구독 플랜 관리 |

---

## 사용자 플로우

1. 로그인 → 홈
2. 홈 → 사건 목록 → 사건 에디터 (서면 작성)
3. 홈 → 증거 목록 → 증거 PDF / 증거 뷰어
4. 홈 → AI 서면 생성

---

## 주요 제한사항

- 비로그인 상태에서 모든 내부 화면 접근 불가 → 로그인 페이지로 리다이렉트
- 구독 만료 시 AI 기능 제한
