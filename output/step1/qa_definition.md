## 로그인 (https://staging.ailex.co.kr)

### 정상 상태 정의
- 이메일/비밀번호 입력 후 로그인 버튼 활성화
- 로그인 성공 시 홈 화면으로 이동

### 의도된 제한사항
- 잘못된 이메일/비밀번호 입력 시 오류 메시지 표시
- 5회 실패 시 계정 잠금

### 비정상 케이스
- 네트워크 오류 시 오류 토스트 표시
- 서버 오류(500) 시 오류 메시지 표시

### 성공 기준
- 유효한 계정으로 로그인 후 홈 화면(`/home`) 이동 확인

### 실패 기준
- 로그인 버튼 클릭 후 10초 내 응답 없음
- 로그인 성공 후 홈이 아닌 화면으로 이동

## 홈 (https://staging.ailex.co.kr/home)

### 정상 상태 정의
- 최근 사건 목록 표시
- 대시보드 통계 수치 표시

### 의도된 제한사항
- 사건이 없을 경우 빈 상태 메시지 표시

### 비정상 케이스
- 데이터 로딩 실패 시 오류 상태 표시

### 성공 기준
- 로그인 후 홈 화면 정상 렌더링 확인

### 실패 기준
- 흰 화면 또는 콘솔 에러 발생

## AI (/ai)

### 정상 상태 정의
- AI 서면 생성 인터페이스 표시
- 구독 활성 상태에서 모든 기능 활성화

### 의도된 제한사항
- 구독 만료 시 기능 제한 (확인 필요)

### 비정상 케이스
- API 응답 지연 시 로딩 인디케이터 표시

### 성공 기준
- 구독 활성 상태에서 AI 기능 정상 동작

### 실패 기준
- 구독 활성임에도 기능 비활성화 표시
